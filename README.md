# Jerry-chine Gun

Bring the chaos of Hypixel SkyBlock's legendary Jerry-chine Gun to your own Minecraft server.
Jerry-chine Gun is a fan-made Paper plugin inspired by the iconic weapon from Hypixel SkyBlock. Equip the weapon, unleash explosive projectiles, and cause pure Jerry-powered destruction wherever you go.

> This project is a fan creation and is not affiliated with, endorsed by, or associated with Hypixel.

---

## Features

* Custom Jerry-chine Gun item
* Explosive projectile launching
* Fun and unique gameplay mechanics
* Lightweight and optimized
* Easy setup and configuration
* Permission-based command system
* Modern Paper support
* Multiplayer friendly

---

## Commands

| Command              | Description                                |
| -------------------- | ------------------------------------------ |
| `/giveitem`          | Gives yourself a Jerry-chine Gun.          |
| `/giveitem <player>` | Gives a Jerry-chine Gun to another player. |

---

## Permissions

| Permission                      | Description                                                 |
| ------------------------------- | ----------------------------------------------------------- |
| `jerrychinegun.giveitem`        | Allows players to receive a Jerry-chine Gun for themselves. |
| `jerrychinegun.giveitem.others` | Allows players to give a Jerry-chine Gun to other players.  |

---

## Installation

1. Download the latest release.
2. Place the plugin JAR inside your server's `plugins` folder.
3. Start or restart your server.
4. Use `/giveitem` to obtain your Jerry-chine Gun.
5. Have fun causing chaos.

---

## Requirements

* Java 25 or newer
* Paper 26.1.2+
* Compatible Paper-based forks
* NOT compatible with Folia at this time

---

## Usage

After obtaining the Jerry-chine Gun, simply hold the item and use it to launch explosive projectiles.
Server owners can control access through the provided permission nodes.
